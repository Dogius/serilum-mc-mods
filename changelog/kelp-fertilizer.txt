Changelog:
Kelp Fertilizer
https://www.curseforge.com/minecraft/mc-mods/kelp-fertilizer

=========
v1.8: (Forge 1.18.1 only) Fixed mod not initializing correctly due to missing file.

=========
v1.8: (Fabric only) Fixed kelp not working correctly on crops.

=========
v1.7: Added swing animation on kelp use.

=========
v1.6: (1.17.1 only) Fixed pack_format integer. It now correctly states 7 instead of 6, which allows the use of datapacks.

=========
v1.5: Updated source code to work with the official Mojang mappings, in preparation for 1.17 and future mod updates. Does not contain any in-game changes.

=========
v1.4: Updated Kelp Fertilizer to work with the new Collective update. The update attempts to inform players which download mods from illegitimate sources that it can be harmful to do so. If you are reading this on CurseForge and download your mods here, this won't affect you. Thank you for downloading mods from CurseForge and supporting the mod developers.

=========
v1.3: Fixed error during complete event phase due to mod incompatibility.

=========
v1.2: Kelp can now be used from the offhand as well as the main hand. Fixed the kelp itemstack being decreased when the player was in creative mode.

=========
v1.1: You can now put kelp inside dispensers to use the bone meal effect.
Added dependency on the Collective library mod.

=========
v1.0: Initial release of mod.
