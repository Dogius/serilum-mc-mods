Changelog:
Configurable Extra Mob Drops
https://curseforge.com/minecraft/mc-mods/configurable-extra-mob-drops

=========
v1.2: (Fabric 1.19 only) Updated mod to work with Minecraft's latest 1.19 release version.

=========
v1.9: (Forge 1.18.1 only) Fixed mod not initializing correctly due to missing file.

=========
v1.8: Fixed mobs with the Misc category not being shown in the mob drop list. Specifically Iron/Snow Golems and Villagers.

=========
v1.7: The mod now correctly parses all characters in item names, like ¢.

=========
v1.6: Updated source code to work with the official Mojang mappings, in preparation for 1.17 and future mod updates. Does not contain any in-game changes.

=========
v1.5: Added support to the cemd command for modded entities with the same name.

=========
v1.4: Fixed the command not choosing the right entity from the input.
Fixed drops not being able to stack with others due to an nbt tag.
Added itemstack amount in confirmation message.

=========
v1.3: Updated Configurable Extra Mob Drops to work with the new Collective update. The update attempts to inform players which download mods from illegitimate sources that it can be harmful to do so. If you are reading this on CurseForge and download your mods here, this won't affect you. Thank you for downloading mods from CurseForge and supporting the mod developers.

=========
v1.2: Improved drop chance handling. Fixes issue with multiple drops.

=========
v1.1: Added the ability to add a drop chance.

=========
v1.0: Initial release of mod.
